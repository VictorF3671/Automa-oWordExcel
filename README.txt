🚨 Avisos Importantes para o Usuário 🚨
* Formato da Planilha

As colunas da planilha devem ter exatamente o mesmo nome que os marcadores do modelo Word (por exemplo, se no modelo você usa {{ nome }}, a coluna deve ser nome no Excel).

Não utilize espaços ou símbolos especiais nos nomes das colunas.

* Conteúdo das Células

Certifique-se de que os valores não têm símbolos adicionais, como R$, vírgulas, pontos, etc. Use apenas números simples para facilitar o preenchimento correto no Word.

* Modelo Word

O modelo deve conter os mesmos marcadores entre chaves ({{ }}) que correspondam aos nomes das colunas da planilha. Por exemplo, se a planilha tem a coluna nome, o Word deve ter o marcador {{ nome }}.

* Diretório de Salvamento

O caminho do diretório de saída deve existir previamente. O sistema não cria o diretório automaticamente.

* Personalização do Nome do Arquivo

No campo “Formato do nome do arquivo”, utilize apenas os nomes das colunas entre chaves, como {nome}, para que o sistema consiga substituir corretamente.
